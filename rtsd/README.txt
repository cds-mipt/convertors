В наборе данных RTSD есть фаил со всеми аннотациями full-gt.csv и архив со всеми изображениями. Кроме того
авторы подготовили поднаборы в detection с данными для обучения и тестирования. Чтобы удобно работать как с
полным архивом, так и с его подмножествами, лучше сначала преобразовать все данные доступные в full-gt.csv и
full-frames, а именно сопоставить классу знака его id, изображению его id, аннотации его id, и сохранить полученный
coco-gt фаил. Этот фаил будет содержать все аннотации, изображения и категории.
Эти операции делает скрипт convert_full.py.

Второй скрипт select.py берет полученный первым скриптом фаил, директорию с картинками и csv фаил, и записывает json,
содержащий только картинки из директории, а аннотации из csv. Также есть возможность оставить только те категирии,
которые есть в csv фаиле или подгрузить категории из другого json фаила. Это удобно при конвертации train/test пар.
